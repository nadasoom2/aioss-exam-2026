# 문항 1 성공 기준 체크리스트

> 아래 항목을 모두 충족해야 문항 1의 점수가 인정됩니다.

## 체크리스트

- [ ] **PR 링크 1개를 제출했다.**
  - `feature/workflow-setup` 브랜치에서 `main`으로 향하는 PR 생성

- [ ] **브랜치 전략 설명을 5문장 이내로 정리했다.**
  - `BRANCH_STRATEGY.md` 파일 상단 5문장 참고

- [ ] **리뷰 코멘트 1개 이상을 남겼다.**
  - PR에서 직접 코드 또는 파일에 리뷰 코멘트 작성

---

## 제출 방법

1. 이 저장소를 Fork 또는 로컬에서 작업
2. `feature/workflow-setup` 브랜치 생성
3. 아래 파일들을 추가하고 커밋

```
BRANCH_STRATEGY.md
REVIEW_CHECKLIST.md
SUCCESS.md
.github/PULL_REQUEST_TEMPLATE.md
```

4. `main` 브랜치로 PR 생성
5. PR에 리뷰 코멘트 1개 이상 작성
6. PR 링크 제출
